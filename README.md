# ChopChopNomNom

ChopChopNomNom is your personal weekly meal planner.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Weekly Planning**: Drag and drop dishes to plan your week.
- **Dish Management**: Add, edit, and organize your favorite recipes.
- **Shopping List**: Automatically generate a shopping list based on your plan.
